Parcours utilisateur de AIsoft :


1. Authentification

L'utilisateur se connecte via /login ou s'inscrit via /signup.

2. Téléversement de l'image via /upload

Il remplit un formulaire avec des infos facultatives (nom, type, marque, prix…)

Il télécharge une photo d’arme airsoft.

L’image est stockée et ajoutée à la base Mongo.

3. Traitement de l’image via /analyze :

L’utilisateur améliore l’image (niveaux de gris, nettoyage, etc.)

Le système tente un matching simple avec la base Mongo 

Extraction des caractéristiques (features) de l’image uploadée → img_resized ou flattened_features

Pour chaque arme dans la base : récupérer les features db_features (par exemple weapon["image_features"])

Calculer la distance entre les deux vecteurs de features (diff)

Trouver l’arme avec la distance minimale

Si cette distance est inférieure à ton seuil → match

Afficher le nom/modèle/marque associés au meilleur match trouvé

➕ Si correspondance trouvée → identification immédiate.
